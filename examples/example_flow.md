# Beispielablauf

Ein vereinfachter Ablauf könnte so gelesen werden:

1. Ein Eingangssignal trifft ein.
2. Das Signal wird formal geprüft.
3. Schutz- und Zuständigkeitslogik greifen.
4. Eine Verarbeitungseinheit übernimmt den Fall.
5. Zustand und Ergebnis werden getrennt behandelt.
6. Eine Ausgabe wird vorbereitet.

Dieser Ablauf ist absichtlich abstrahiert und nicht identisch mit einer produktiven Implementierung.
