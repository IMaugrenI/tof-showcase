# Repository role

This repository is the primary public entry point for the public ToF repository family.

## Why this file exists

Public readers should not have to guess which repository to open first.
This file makes the entry hierarchy explicit.

## Reading order

1. Start here in `tof-showcase`.
2. Use this repository to understand the public architectural frame, layer separation,
   and boundary language.
3. Move to narrower or more product-shaped repositories only after that:
   - `tof_local_builder`
   - `tof_local_knowledge`
   - `tof-v7-public-frame`

## Public position

This repository is a controlled public frame.
It is not the private runtime, not the full production truth, and not a hidden dump.

## About V’eth in public

V’eth is referenced publicly in this repository only as a separate identity, mirror,
expression, and symbolic layer.

That public naming does not imply that private canon, runtime-sensitive material,
or operator logic is publicly disclosed here.

## Cross-repo rule

Links to other repositories are navigational.
They are not a statement that all repositories share the same legal terms or the same exposure depth.
Always read each repository on its own terms.
